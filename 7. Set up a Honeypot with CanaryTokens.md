
# 7. Set up a Honeypot with CanaryTokens

<br>

<p> A canary token is a tool used to monitor certain resources, such as files, URLs, or API keys, for access. If any of these resources are accessed, an alert is sent to the owner of the resource to notify them of the access.

In numerous reports on ransomware attacks, it is common to observe the attackers executing a set of predictable commands on endpoints. Below are some examples 
 ``` 
whoami
ipconfig /all
tasklist 
systeminfo
wmic product get name,version
wmic /node:<REDACTED> process list brief
net view \\<REDACTED>\Files$ /all
dir \\<REDACTED>\C$\   
  ```
 <p> CanaryTokens can aid in letting us detect when these commands have been used, This works best for small organizations that don't have the resources of a large orginizations. </p>
