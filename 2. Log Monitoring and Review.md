
# 2. Log Monitoring and Review

<br>

##  1. Access the Event Viewer Logs

<p align="left">
   Note: There are various types of event logs, including application logs, system logs, and security logs. While application and system logs can be accessed by all users, only administrators are able to view security logs <br>
</p>

<br>

<p align="center">
   Open the Event Viewer <br>
  <img src="https://user-images.githubusercontent.com/121813491/211128870-a51e3c1a-be60-45ac-a1bc-2fad39b9c625.jpg" alt="Event Viewer console "  width="60%" height="60%">
</p>
<p align="left">
  
  <br>
  
  
  Note: The Event Viewer console is displayed, and it includes several types of logs:

    Application: Events generated by applications
    Security: Security-related events generated by the OS
    Setup: Events related to application installation
    System: Events generated by the OS
  </p>

<br>


<p align="center">
   After expanding the Windows Logs node, and then select the Application log <br>
  <img src="https://user-images.githubusercontent.com/121813491/211128960-6f827690-deea-4c90-9fd5-a8f061e04724.jpg" alt="Application log"  width="60%" height="60%">
</p>

<br>


<p align="left">
          The Windows Logs node displays various types of logs that are static on all Windows systems. However, a server that has specific services like a DNS server can also have DNS logs under the Applications and Services Logs node
   </p>

## 2. SIEM Logs(ALIEN VAULT OSSIM)

<p align="left">
Alien Vault OSSIM is a security application suite that includes a Host Intrusion Detection System (HIDS) and a built-in SIEM tool for monitoring devices
  </p>
  
 <br>


  <p align="center">
   Log into ALIEN VAULT OSSIM <br>
  <img src="https://user-images.githubusercontent.com/121813491/211129787-6ecab384-50e8-4c0d-b854-0e8dca013964.png" alt="ALIEN VAULT OSSIM Login"  width="60%" height="60%">
</p>
  
  <br>
  
<p align="center">
   The Alien Vault Dashboard is visible, and from there you can go to the ENVIRONMENT tab and select DETECTION <br>
  <img src="https://user-images.githubusercontent.com/121813491/211152814-556d853b-8353-4946-b9b5-999c04551ead.jpg" alt="ALIEN VAULT OSSIM Dashboard"  width="60%" height="60%">
</p>
  
 <br>
 
 <p align="center">
   To add an agent, go to the DETECTION page and click 'AGENTS', then click ADD AGENT <br>
  <img src="https://user-images.githubusercontent.com/121813491/211129956-df40ba58-8df2-4307-b4a4-48aecc7ce886.jpg" alt="DETECTION page"  width="60%" height="60%">
</p>

<br>

 <p align="center">
   In the "NEW HIDS AGENT" dialog box, enter the IP address or name into the "Search by IP address or name" text box <br>
  <img src="https://user-images.githubusercontent.com/121813491/211152761-34119748-6385-4a09-a95b-4bdba83bcab5.jpg" alt="NEW HIDS AGENT dialog box"  width="60%" height="60%">
</p>

<br>

 <p align="center">
   In the "To deploy the HIDS (Host-based Intrusion Detection System) agent, you have the option to either do it manually by creating an installer or automatically. In this case, the HIDS agent will be deployed automatically by clicking the "Automatic HIDS deployment for Windows" icon, which is located on the far right <br>
  <img src="https://user-images.githubusercontent.com/121813491/211153223-393d6183-c93c-4493-abe2-72f9f6fc3069.jpg" alt="Automatic HIDS deployment for Windows"  width="60%" height="60%">
</p>

<br>

 <p align="center">
  A dialog box titled 'AUTOMATIC DEPLOYMENT FOR WINDOWS' will appear. You need to enter information in the text boxes provided <br>
  <img src="https://user-images.githubusercontent.com/121813491/211153512-dbfeed4a-8ec0-46f6-9d72-583a8cb5268b.jpg" alt="Automatic HIDS deployment for Windows"  width="60%" height="60%">
</p>

<br>

<p align="center">
    On the 'SECURITY EVENTS (SIEM)' page, choose 'AlientVault HIDS' from the 'DATA SOURCES' drop-down menu. You will see it added to the text box below. Scroll down and you will see events that have been captured from the IP address you specified for the scan. The SIEM has registered events from the HIDS that were installed on the IP address you entered
  <img src="https://user-images.githubusercontent.com/121813491/211153643-ab56551e-92dd-4e76-9bf5-5dc35c2b8fa8.jpg " alt="Automatic HIDS deployment for Windows"  width="60%" height="60%">
